# History App

## Local testing

How to start the app

1. `npm start`
2. Go to <http://localhost:3000/> in your web browser, if it doesn't take you there automatically.

## Deploying

How to deploy to <https://afx.dance/>.

1. `npm run build`
2. `git add docs`
3. `git commit -m "<sample message>"`
4. `git push`
